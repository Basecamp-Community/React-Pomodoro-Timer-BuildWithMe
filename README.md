## Getting Started with Project
1. Clone the Repo
```bash
git clone https://github.com/Basecamp-Community/React-Pomodoro-Timer-BuildWithMe.git

cd React-Pomodoro-Timer-BuildWithMe
```
2. Install Dependencies
```bash
npm install
```
3. Start the Local Server
```bash
npm start
```
